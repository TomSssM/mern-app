## Quick start

Use the requisite Node.js version:

```bash
nvm use
```

Install dependencies:

```bash
npm ci
```

## Development

Start development server:

```bash
npm run dev
```

Server runs on `http://localhost:3000` and client on `http://localhost:8080`

Lint the project:

```bash
npm run lint
```

Fix lint errors in the project:

```bash
npm run fix
```

Start storybook:

```bash
npm run storybook
```
